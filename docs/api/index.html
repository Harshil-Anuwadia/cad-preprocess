API Reference

Complete API documentation for the CAD Preprocess library.

Package Structure

cad_preprocess/
├── __init__.py           # Package exports
├── integration.py        # High-level API (CADPreprocessor, preprocess)
├── input_handler.py      # DICOM discovery and validation
├── preprocessing_engine.py  # Image processing pipeline
├── metadata_extractor.py # Metadata extraction
├── output_writer.py      # File persistence
├── config.py             # Configuration management
├── logging_utils.py      # Logging and error handling
└── cli.py                # Command-line interface

Quick Import Reference

# High-level API (recommended)
from cad_preprocess import preprocess, CADPreprocessor

# Configuration
from cad_preprocess import PreprocessingConfig

# Individual modules
from cad_preprocess import (
    InputHandler,
    PreprocessingEngine,
    MetadataExtractor,
    OutputWriter
)

# Logging utilities
from cad_preprocess.logging_utils import (
    ProcessingStats,
    ProcessingError,
    fail_safe,
    setup_logging
)

Module Overview

Module Description Key Classes/Functions
integration High-level API for common workflows CADPreprocessor, preprocess()
input_handler DICOM file discovery and validation InputHandler, DicomFile
preprocessing_engine Image preprocessing pipeline PreprocessingEngine
metadata_extractor DICOM metadata extraction MetadataExtractor
output_writer File output and persistence OutputWriter
config Configuration management PreprocessingConfig
logging_utils Logging and error handling ProcessingStats, fail_safe
cli Command-line interface main(), parse_args()

Type Definitions

ProcessingResult

class ProcessingResult:
    sop_instance_uid: str      # Unique identifier
    image: np.ndarray          # Preprocessed image array
    metadata: Dict[str, Any]   # Extracted metadata
    original_shape: Tuple[int, int]  # Original dimensions
    final_shape: Tuple[int, int]     # Final dimensions
    modality: str              # DICOM modality
    processing_time: float     # Time in seconds

ProcessingStats

class ProcessingStats:
    total: int        # Total files found
    processed: int    # Successfully processed
    failed: int       # Failed to process
    skipped: int      # Skipped (invalid DICOM)
    duration: float   # Total time in seconds

Exceptions

Exception Description
ProcessingError Base exception for processing errors
DicomReadError Failed to read DICOM file
ConfigurationError Invalid configuration
OutputError Failed to write output file
