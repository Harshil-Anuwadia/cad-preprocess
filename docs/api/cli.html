Programmatic Usage

The CLI module can be used programmatically:

Run CLI from Python

from cad_preprocess.cli import main

# Run with arguments
exit_code = main(["-i", "./dicoms", "-o", "./output"])

# With config file
exit_code = main([
    "-i", "./dicoms",
    "-o", "./output",
    "-c", "config.yaml"
])

Parse Arguments Only

from cad_preprocess.cli import parse_args

args = parse_args(["-i", "./dicoms", "-o", "./output"])
print(args.input)   # ./dicoms
print(args.output)  # ./output

Get Help Text

from cad_preprocess.cli import create_parser

parser = create_parser()
help_text = parser.format_help()
print(help_text)

Exit Codes

Code Constant Meaning
0 EXIT_SUCCESS Success
1 EXIT_ERROR General error
2 EXIT_INVALID_ARGS Invalid arguments
3 EXIT_INPUT_NOT_FOUND Input path not found
4 EXIT_NO_FILES No DICOM files found 
from cad_preprocess.cli import main, EXIT_SUCCESS

exit_code = main(["-i", "./dicoms", "-o", "./output"])
if exit_code == EXIT_SUCCESS:
    print("Processing completed successfully")
