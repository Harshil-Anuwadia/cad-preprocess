PreprocessingEngine

Core preprocessing engine that applies deterministic transformations to DICOM images.

class PreprocessingEngine(config=None, **kwargs)

Constructor Parameters

config PreprocessingConfig | None
Configuration object
target_size int | Tuple[int, int] | None
Target output size
normalize bool
Normalize to 0-255 (default: True)
window_center float | None
CT window center (None = use DICOM value)
window_width float | None
CT window width (None = use DICOM value)
interpolation str
Resize interpolation: "nearest", "bilinear", "bicubic"

Example

from cad_preprocess import PreprocessingEngine
import pydicom

# Create engine
engine = PreprocessingEngine(
    target_size=512,
    normalize=True,
    window_center=40,
    window_width=400
)

# Process DICOM
dcm = pydicom.dcmread("ct_image.dcm")
processed = engine.process(dcm)

print(processed.shape)  # (512, 512)
print(processed.dtype)  # uint8

üìù Determinism
All operations in PreprocessingEngine are deterministic. The same input with the same configuration will always produce identical output.
