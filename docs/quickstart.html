Basic Usage

CAD Preprocess can be used in two ways:

  1. Command Line Interface (CLI) - For quick processing and scripts
  2. Python API - For integration into your applications

CLI Examples

Process a Single DICOM File

cad-preprocess -i image.dcm -o ./output

Process a Directory of DICOM Files

cad-preprocess -i ./dicoms -o ./output

Use a Configuration File

cad-preprocess -i ./dicoms -o ./output -c config.yaml

Preview Without Processing (Dry Run)

cad-preprocess -i ./dicoms -o ./output --dry-run

This shows what would be processed without actually creating any output files.

Specify Metadata Profile

# Extract ML-focused metadata
cad-preprocess -i ./dicoms -o ./output --metadata-profile ml

# Extract minimal metadata only
cad-preprocess -i ./dicoms -o ./output --metadata-profile minimal

Custom Window Settings (CT Images)

# Lung window
cad-preprocess -i ./dicoms -o ./output --window-center -600 --window-width 1500

# Bone window
cad-preprocess -i ./dicoms -o ./output --window-center 400 --window-width 1800

Resize Output Images

cad-preprocess -i ./dicoms -o ./output --target-size 512

Python API Examples

Simple One-Liner

from cad_preprocess import preprocess

# Process a single file
result = preprocess("image.dcm", "output/")
print(f"Processed: {result.sop_instance_uid}")
print(f"Shape: {result.image.shape}")

Batch Processing

from cad_preprocess import CADPreprocessor

# Create processor
processor = CADPreprocessor()

# Process entire directory
results = processor.process_directory("dicoms/", "output/")

print(f"Processed {len(results)} files")
for result in results:
    print(f"  - {result.sop_instance_uid}")

Using Configuration

from cad_preprocess import CADPreprocessor

# Load from config file
processor = CADPreprocessor.from_config("config.yaml")

# Or configure programmatically
processor = CADPreprocessor(
    target_size=512,
    normalize=True,
    metadata_profile="ml"
)

results = processor.process_directory("dicoms/", "output/")

Reproducibility with Config Hash

from cad_preprocess import CADPreprocessor

# Training: save the config hash
processor = CADPreprocessor.from_config("config.yaml")
config_hash = processor.config_hash
print(f"Config hash: {config_hash}")

# Inference: verify same configuration
inference_processor = CADPreprocessor.from_config("config.yaml")
assert inference_processor.config_hash == config_hash

Access Processing Statistics

from cad_preprocess import CADPreprocessor

processor = CADPreprocessor()
results = processor.process_directory("dicoms/", "output/")

# Get statistics
stats = processor.stats
print(f"Processed: {stats.processed}")
print(f"Failed: {stats.failed}")
print(f"Skipped: {stats.skipped}")
