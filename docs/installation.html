Requirements

CAD Preprocess requires the following:

Requirement Version Notes
Python >= 3.9 Python 3.9, 3.10, 3.11, or 3.12
pydicom >= 2.3.0 DICOM file reading
numpy >= 1.21.0 Array operations
Pillow >= 9.0.0 Image I/O
PyYAML >= 6.0 Configuration files
scikit-image >= 0.19.0 Image processing

Install from Source

The recommended way to install CAD Preprocess is from the source repository:

# Clone the repository
git clone https://github.com/Harshil-Anuwadia/cad-preprocess.git
cd cad-preprocess

# Install in editable mode
pip install -e .

This installs the package in "editable" mode, meaning any changes to the source code will be reflected immediately without reinstalling.

Install with pip

You can also install the dependencies first, then install the package:

# Install dependencies
pip install pydicom numpy Pillow PyYAML scikit-image

# Clone and install
git clone https://github.com/Harshil-Anuwadia/cad-preprocess.git
cd cad-preprocess
pip install -e .

Virtual Environment (Recommended)

It's recommended to use a virtual environment to avoid conflicts with other packages:

Using venv

# Create virtual environment
python -m venv .venv

# Activate (Linux/macOS)
source .venv/bin/activate

# Activate (Windows)
.venv\Scripts\activate

# Install the package
pip install -e .

Using conda

# Create conda environment
conda create -n cad-preprocess python=3.11

# Activate
conda activate cad-preprocess

# Install the package
pip install -e .

Verify Installation

After installation, verify that everything is working:

Check CLI

cad-preprocess --version
# Output: cad-preprocess 0.1.0

cad-preprocess --help

Check Python Import

python -c "from cad_preprocess import preprocess; print('Success!')"

Run Tests

# Install pytest if needed
pip install pytest

# Run all tests
pytest tests/

Troubleshooting

ImportError: No module named 'pydicom'

Make sure all dependencies are installed:

pip install pydicom numpy Pillow PyYAML scikit-image

Command 'cad-preprocess' not found

Ensure you installed the package and your virtual environment is activated:

# Check if installed
pip list | grep cad-preprocess

# Reinstall if needed
pip install -e .

Python version error

CAD Preprocess requires Python 3.9+. Check your version:

python --version
