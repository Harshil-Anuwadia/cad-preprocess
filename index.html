CAD Preprocess Documentation

CAD Preprocess is a Python library for DICOM preprocessing in CAD (Computer-Aided Detection/Diagnosis) systems. It standardizes medical image preprocessing so it doesn't need to be rewritten across training, inference, and UI workflows.

Quick Example

Command Line

# Process a DICOM directory
cad-preprocess -i ./dicoms -o ./output

# With configuration file
cad-preprocess -i ./dicoms -o ./output -c config.yaml

# Extract ML-focused metadata
cad-preprocess -i ./dicoms -o ./output --metadata-profile ml

Python API

from cad_preprocess import preprocess, CADPreprocessor

# Simple one-liner
result = preprocess("image.dcm", "output/")
print(f"Shape: {result.image.shape}")

# Batch processing with configuration
processor = CADPreprocessor.from_config("config.yaml")
batch = processor.process_directory("dicoms/", "output/")

# Store config hash for reproducibility
config_hash = processor.config_hash

Requirements

  • Pillow >= 9.0.0
  • PyYAML >= 6.0
  • scikit-image >= 0.19.0
